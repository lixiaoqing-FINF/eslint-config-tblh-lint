# eslint-config-tblh-lint

### Install
If using npm 5+, use this shortcut
```js
npx install-peerdeps --dev eslint-config-tblh-lint
```
### How To Use
```js
add ' "extends": ["tblh-lint"] ' to your .eslintrc
```
